## 构建适用于本机的 Clash 命令行版

1. 下载上方列出的 Clash 源码压缩包后，解压
2. 安装 Go 运行时
3. 在源码解压目录内，运行命令：`go build ./`
4. 目录内的 `Clash` 或 `Clash.exe` 即为适用于本机的 Clash 命令行版
